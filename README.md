# Hanan Technologies – Stamp & Printing Solutions

This repository contains the marketing website and product catalogue for **Hanan Technologies**, a premium provider of stamp and printing solutions in Dubai.

## Getting started

**Prerequisites**

- Node.js and npm installed

**Install dependencies**

```sh
npm install
```

**Run the development server**

```sh
npm run dev
```

Then open `http://localhost:5173` in your browser.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
